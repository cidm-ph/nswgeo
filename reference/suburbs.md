# Suburbs of New South Wales.

A dataset containing the names of suburbs in NSW and their postcodes.
These fields are extracted as-is from the source dataset published by
DCS Spatial Services, NSW Government.

## Usage

``` r
suburbs
```

## Format

A data frame with 2 columns:

- suburbname:

  The name of the suburb, in upper case

- postcode:

  The main postcode of the suburb, as a character
